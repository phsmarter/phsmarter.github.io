To determine who got the best deal, we need to calculate the cost-to-book ratio for each person. This is done by dividing the total cost by the number of books purchased.

Jo's ratio is 9/4 = 2.25

Kei's ratio is 12/8 = 1.5

Kate's ratio is 30/16 = 1.875

Bryn's ratio is 8/4 = 2
