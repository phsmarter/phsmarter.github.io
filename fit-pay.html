2. A fitness center has a swimming pool and a gym. There are 3,924 members in the fitness club. There are two kinds of membership: regular and VIP . Each regular pays $25 per month and each VIP member pays $480 per year. how much in membership fees does the fitness center receive from the regular members each month??​


Answer: the fitness center would receive 3924 * $25 = $98,100 from the regular members each month.

Step-by-step explanation:

To find the membership fees received from the regular members each month, we can multiply the number of regular members by the monthly fee of $25. Assuming that all 3,924 members belong to the fitness club, we have:

Number of regular members = 3,924 - (Number of VIP members)

Membership fees from regular members = Number of regular members * $25

Membership fees from regular members = (3,924 - Number of VIP members) * $25

So, the fitness center would receive 3924 * $25 = $98,100 from the regular members each month.

15. At a local fitness​ center, members pay an $8 membership fee and ​$3 or each aerobics class. Nonmembers pay ​$4 for each aerobics class. For what number of aerobics classes will the cost for members and nonmembers be the​ same?


Answer:

There are 8 aerobic classes where the total cost to be paid by members and non-members is equal.

Step-by-step explanation:

Let f(x) represent the cost for members:

f(x) = 8 + 3x

Where:

8 ⇒ is the one-time membership fee

3 ⇒ the additional amount to be paid per aerobic class

x ⇒ the number of aerobic class

Let g(x) represent the cost for non-members:

g(x) = 4x

Where:

4 ⇒ the amount paid for each aerobic class

x ⇒ the number of aerobic class

To find the number of class where the costs are equal for both members and non-members:

Cost for members: f(x)

Cost for non-members: g(x)

Equation:

f(x) = g(x)

8 + 3x = 4x

Solution:

4x - 3x = 8

x = 8 ⇒ total number of aerobic classes

The total number of aerobic class where the total cost is equal is 8 classes.

Check:

Cost for members for 8 aerobic classes:

f(x) = 8 + 3x

f(8) = 8 + 3(24)

f(8) = 8 + 24

f(8) = $32

Total cost for 8 aerobic classes for non-members:

g(x) = 4x

g(8) = 4(8)

g(8) = $32

Therefore:

f(8) = g(8)

$32 = $32  (True)
