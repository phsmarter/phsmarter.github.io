11. if one letter is chosen at random from the word assists, what is the probability that the letter chosen will be an "s" ​


Answer:

The probability that the letter chosen will be an "s" is ‌4⁄7‌ or 0.57 or 57%.

Step-by-step explanation:

Probability

Probability is a numerical description of how likely an event is to occur or how likely it is to happen. Probability does not tell us exactly what will happen, it is just a guide.

To find the probability that the letter chosen will be an "s" from the word assists, we have the formula:

Probability

Probability= number of events to happen / total number of possible outcomes

Three Ways to Express Probability

We can express our answer in three ways:

Fraction Form (⅔)Decimal Form (0.67)Percentage Form (67%)

In the given situation above, the number of events to happen is 4 because there are four letter "s" from the word assists. Then, the total number of possible outcomes is 7 because the word assists has seven letters in all.

Solution:

Substitute the values.

Probability

= number of events to happen / total number of possible outcomes

= ‌4⁄7‌

We can also express this as 0.57 or 57%.

Final Answer:

The probability that the letter chosen will be an "s" is ‌4⁄7‌ or 0.57 or 57%.

24. Application software that assist users in regular office jobs. *Your answer​


Office Tools

Office tools are pieces of application software that help users complete common office tasks including writing, editing, and maintaining documents, managing vast quantities of data, producing presentations, scheduling, etc. Using office supplies allows you to do many repetitive chores quickly and with little effort. Among the programs that can achieve this are:

Spreadsheets

Software called a spreadsheet helps users handle and analyze tabular data. It is a tool for electronic bookkeeping. Data is always input into cells, which are formed by the intersection of a row and a column, and it is simple to use formulae and functions to process a collection of cells. Microsoft Excel, Gnumeric, Google Sheets, and others are some of the most well-known spreadsheet programs.

The following are some of the things that can be done with spreadsheet software are straightforward computations, such as addition, averaging, counting, etc.; creating graphs and charts for a set of connected data; input of data; Formatting data; formatting cells; logical comparison-based calculations.

Word processors

Word processing software is used to create, save, and edit text documents. Word processors that are widely used include MS-Word, WordPad, WordPerfect, Google Docs, etc.

It is possible to create, save, and modify documents using a word processor; Format text by changing its font, alignment, color, and background; Check your grammar and spelling; Add pictures, headers, and footers, as well as watermarks and page margins.

Databases systems

Database management systems are pieces of software that control how data is stored, updated, and retrieved using databases. Tools for managing databases include MS-Access, MySQL, Oracle, FoxPro, and others.

Presentation tools

With the use of a presentation tool, users may present material that has been divided up into manageable pieces and set up on slides. A presentation is a collection of slides that clearly communicate a concept to the audience. Text, photos, tables, music, video, and other multimedia content can all be organized on the slides. Popular presenting programs include MS-PowerPoint, OpenOffice Impress, Lotus Freelance, and others.
