9. A(n) ____________________-based IDPS resides on a particular computer or server and monitors activity only on that system.


A host based IDS resides on a particular computer or server and monitors activity only on that system.

Explanation :

Host-Based Intrusion Detection System (HIDS) is a system capable of detecting and identifying threats that occur in computer systems, by recognizing any pattern of attacks carried out by intruders.

Host-based IDS tends to be similar to network-based. The difference lies in that Host-Based IDS is designed to monitor networks and computers, while network-based IDS is only designed to monitor network traffic.

The example of host-based IDS is when IDS monitors system registry changes such as repeated login attempts with incorrect passwords or installation of backdoors. This activity will be indicated as a suspicious sign monitored by the system.
