29. The nervous system controls all the activities of other systems like in running. Below is a game board showing the activities during a Fun Run, identify the primary organ system used in each statement.​


Answer:1 digestive system

2. Nervous system

3. Muscular and skeletal system

4. Respiratory system

5. Excretory system

6.circulatory system

Explanation:

Answer:

1. Digestive system — digestive system is responsible for breaking down nutrients from the food we eat.

2. Nervous system — a sudden feel of excitement is a reponse from the hypothalamus which activates the sympathetic nervous system, a central for emptional reaction.

3. Musculoskeletal — muscles and bones work together to perform specific action.

4. Respiratory system — respiratory system is responsible for two processes, taking in oxygen and releasing carbon dioxide. In other words, responsible for breathing or respiration.

5. Integumentary system — sweat production is a function of the integumentary system, this is to maintain the proper body temperature.

6. Circulatory system - the sudden increase of heart rate is expected after you run because the heart keeps pumping the blood, so that the oxygen and nutrients will be deliver to the different parts of the body which is needed to keep the body going.
