3. What happens to the value of speed if the distance traveled increases when the time of travel is constant? * 1 point O Value of speed decreases. Value of speed increases. Value of speed remains the same. O Value of speed will be equal to zero.​


Answer:

If the distance traveled increases while the time of travel remains constant, the value of speed will increase. This is because speed is calculated as distance divided by time, and if distance increases while time remains constant, the ratio of distance to time (i.e., speed) will increase.

Answer:

480 km

Step-by-step explanation:

let t1 = time for the boat to travel x distance

    t2 = time for the ocean liner to travel x distance

Keep in mind that the speeding boat reached the x distance faster as it travels faster than the ocean liner.

So, t1 + 2 = t2   "the difference in time taken was 2 hours"

Let's begin with "traveled the same distance" equation.

D_boat = D_liner

      TR  = TR

  t1 (80) = t2 (60)

  t1 (80) = (t1 + 2) (60)

  t1 (4/3) = t1 + 2

t1 (4/3) - t1 = 2

      t1 (1/3) = 2

           t1   =  6 hours

t2 = t1 + 2

t2 = 6+2

t2 = 8hours

let's substitute the values back in the equations.

D_boat = TR

            = 6(80) = 480 km

D_liner = TR

            = 8(60) = 480 km

distance x = 480 km
