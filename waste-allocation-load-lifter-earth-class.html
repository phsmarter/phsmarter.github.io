16. A weight lifter lifts a 60 kg barbbell to a vertical height of 2 m. How much work is done by the weight lifter? (Show your solution.)​


[tex]\tt{\huge{\blue{Explanation:}}}[/tex]

The work done by an object exerting a force F at a distance d is given by

[tex]\boxed{W = Fd}[/tex]

where:

W = work

F = force

d = distance

[tex]\tt{\huge{\red{Solution:}}}[/tex]

Solving for F

F = weight = mg

F = (60 kg)(9.8 m/s²)

F = 588 N

Solving for W

W = Fd

W = (588 N)(2 m)

[tex]\boxed{W = \text{1,176 J}}[/tex]

Therefore, the work done is 1,176 J.
