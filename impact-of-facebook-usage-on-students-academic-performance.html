29. A group of 25 high school student were asked whether they use eigther facebook or twitter or both. Fifteen of these student use facebook and twelve use twitter.


Answer:

2 users use both. 13 users use facebook only and 10 users use twitter only.


Step-by-step explanation:

Total number of students = 25

Number of facebook users = 15

Number of twitter users = 12


Find the number of users that use both facebook and twitter:

Number of users = 15 + 12 - 25 = 2


Find the number of users that use facebook only:

Number of users = 15 - 2 = 13


Find the number of users that use twitter only:

Number of users = 12 - 2 = 10


Answer: 2 users use both. 13 users use facebook only and 10 users use  twitter only.
