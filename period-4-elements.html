29. an element is located in a group 2 period 4. the element is a/an​


Answer:

Group 4 is the second group of transition metals in the periodic table.

It contains the four elements titanium (Ti), zirconium (Zr), hafnium (Hf), and rutherfordium (Rf). The group is also called the titanium group or titanium family after its lightest member.


30. 4. locate the elements in the periodic table;​


Answer:

Using the location code (periodic table coordinate), find the symbol for the element on a periodic table. The First Number identifies the column (vertical group) that the element is in. The Second Number identifies the row (horizontal period) the element is in.
