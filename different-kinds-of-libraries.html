11. There are many books in a library . Library A has 10 384 books while library B has 8294 books. what is the difference in the number of books in library A and library B rounded to the nearest hundred


Answer:

2,100

Step-by-step explanation:

The difference means we have to subtract the number of books in both libraries.

10,384-8,294=2,090.

Since we have to round it to the nearest hundred, the hundred place is the 0 between 2 and 9. The next number after 0 is 9 so we have to round up 0 to 1.

Hence, 2,090 in the nearest hundred is 2,100.

Create instances

After you create a symbol, you can create instances of that symbol throughout your document, including inside other symbols. When you modify the symbol, Animate updates all instances of the symbol.

You can give names to instances from the Property inspector. Use the instance name to refer to an instance in ActionScript. To control instances with ActionScript®, give each instance within a single timeline a unique name.

To specify color effects, assign actions, set the graphic display mode, or change the behavior of new instances, use the Property inspector. The behavior of the instance is the same as the symbol behavior, unless you specify otherwise. Any changes you make affect only the instance and not the symbol.

Editing instance properties

Each symbol instance has its own properties that are separate from the symbol. You can change the tint, transparency, and brightness of an instance; redefine how the instance behaves (for example, change a graphic to a movie clip); and specify how an animation plays inside a graphic instance. You can also skew, rotate, or scale an instance without affecting the symbol.

30. what are the three kinds of materials found in a library ​


Answer:

Libraries typically have a wide variety of materials available for patrons to use. Some of the most common types of materials found in a library are:

1)Books: These are one of the most traditional types of materials found in a library. Books can include fiction and nonfiction, in a variety of formats such as hardback, paperback, and e-books.

2)Periodicals: Periodicals include magazines, journals, newspapers, and other publications that are published on a regular basis. These materials can provide information on current events, research, and other topics of interest.

3)Media: Many libraries also have a variety of media materials available, such as DVDs, CDs, and other audio-visual materials. This can include movies, music, documentaries, and other types of entertainment.

Additionally, Some libraries might have a digital collection, such as e-books, e-audiobooks, and streaming services, some other libraries have special collections, such as rare books, manuscripts, and items of historical importance, the use of some of these materials might need special permission or an appointment.

It's important to note that the specific types of materials available in a library can vary depending on the library and its mission, size, and budget.
