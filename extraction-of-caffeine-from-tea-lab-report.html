Answer:

a.)

C = 49.48%

H = 5%

N = 28.87%

O = 16.49%

b.)

mole of C = 49.48 g / 12 g = 4.12 mol

mole of H = 5 g / 1 g = 5 mol

mole of N = 28.87 g / 14 g = 2.06 mol

mole of O = 16.49 g / 16 g = 1.03 mol

c.)

C = 4.12 mol / 1.03 mol = 4

H = 5 mol / 1.03 mol = 5

N = 2.06 mol / 1.03 mol = 2

O = 1.03 mol / 1.03 mol =  1

d.)

`C4H5N2O

e.)

C = 4*12 = 48 amu

H = 5*1 = 5 amu

N = 2*14 = 28 amu

O = 1*16 = 16 amu

48+5+28+16 = 97 g/mol

f.)

194 g/mol /(divided by) 97 g/mol = 2

g.)

C = 4*2 = 8

H = 5*2 = 10

N = 2*2 = 4

O = 1*2 = 2
