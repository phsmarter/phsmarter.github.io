15. 2. is it possible to run a different operating system that is intended for a specific device, e.g., running an android os on an iphone?


Answer:

The smartphone OS and hardware are typically closely correlated. For one or more phone functions to function, the OS must have drivers for the various phone hardware systems. Because Apple won't release the hardware information the OS writer would need—in fact, it might sue them for trying—putting Android on an iPhone is unlikely to succeed. It might function a little bit better to install iOS on an Android phone; this would be a Hackintosh, but in this case, you would need to reverse-engineer iOS to figure out how to connect it to the hardware you have. Apple dislikes this and will almost certainly not assist you in doing it.

Android is the most open operating system for smartphones; in addition to letting you install custom ROMs (Android versions made by third parties), it may also allow you to install straight Linux, especially if the phone manufacturer was willing to share hardware specifications. It's a bit pointless unless you just want to use the phone as a pocket computer because running Linux on a smartphone (as opposed to Android) prevents you from installing all those Android apps.
