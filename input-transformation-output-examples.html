18. 5 examples of input,5 examples of output and 5 example of storage peripherals.Label each example given ​


Input peripherals:

1. Keyboard - This is a device used to input data into a computer system by typing on the keys. (input)

2. Mouse - This is a device used to control the movement of the cursor on the screen. (input)

3. Scanner - This is a device used to scan and digitize documents and images. (input)

4. Microphone - This is a device used to capture sound and voice input. (input)

5. Webcam - This is a device used to capture video input, often used for video conferencing or live streaming. (input)

Output peripherals:

1. Monitor - This is a display device that shows visual output from a computer system. (output)

2. Printer - This is a device used to print out documents and images from a computer system. (output)

3. Speakers - This is a device used to play audio output from a computer system. (output)

4. Projector - This is a device used to display visual output on a larger screen or surface. (output)

5. Headphones - This is a device used to listen to audio output from a computer system in a private or isolated manner. (output)

Storage peripherals:

1. Hard disk drive - This is a device used to store digital data on a computer system for long-term use. (storage)

2. USB flash drive - This is a small portable device used to store and transfer digital data between different computer systems. (storage)

3. CD-ROM - This is a type of optical disc used for storing digital data such as music, videos, and software applications. (storage)

4. Memory card - This is a small portable device used to store digital data, often used in cameras and mobile devices. (storage)

5. External hard drive - This is a large capacity storage device that is connected to a computer system externally, often used for backup and archiving purposes. (storage)

Output: The result of the processed data is the receipt.
