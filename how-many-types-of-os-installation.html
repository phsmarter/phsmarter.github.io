1. how many os can virtualbox install​


In a VirtualBox, it does not matter how many guest operating system you will have because you can install as many as you can. However, there is a limit depending on the disk space and memory that your computer has. If you run many guest operating systems at the same time, it will slow down in terms of performance when your host operating system has no more free memory to accommodate the guest operating systems.


2. differentiate the two types of OS installation​


Operating systems installation

The Clean Installation  

A totally modern installation of a working system or application on a computer. In a clean introduction of an OS, the hard disk is designed and totally erased. In a clean install of an application, the older form is uninstalled, to begin with. Introducing an OS on a modern computer or introducing an application for the primary time is consequently a clean install. Differentiate with "in-place upgrade."

Additionally, a clean install may be suitable when introducing an OS on an unused hard drive or when exchanging proprietorship of a computer to another individual. Both Windows and Mac OS X permit you to perform a clean install when overhauling your operating system.

The Clean Upgrade  

If the clean install is for a more up-to-date adaptation of the program as of now running on the computer, it is additionally called a "clean update." This is in some cases referred to as migration.
