4. PART II: ESSAY. How can you differentiate operating system from computer applications?Write your answer bellow​


Answer:

The main difference between the operating system and the software of the application is that an operating system is a software system that acts as the user-hardware interface, while the software of the application is a particular task program.

Computer software is separated into software for the system and software for the application. The type of software that is used to run hardware devices is machine software. For machine software, a few examples are operating systems, language processors and application drivers.

Software for applications is the kind of software that is built to fulfill a particular user requirement. Some software applications include word processors, spreadsheets and web browsers. In addition to these forms, there is another program called utility applications. This software assists the tasks of the system. Such examples of utility applications are antivirus software, file and disk management tools.

1. What is purpose of an intermation system from a business perspective? What role does it play in

the business information value chains.

Answer:

What is purpose of an intermation system from a business perspective?

- Business Information System, eases the process of decision making and simplifies the process of delivering the required information and hence assists in taking better decisions instantly. Business Information System can be effectively implemented to help communication better between the employers and the employees.

What role does it play in the business information value chains?

- Information systems help businesses in developing a larger number of value added systems in the company. For example, a company can integrate information systems with the manufacturing cycle to ensure that the output it produces complies with the requirements of the various quality management standards.

From a business perspective, an information system is an important instrument for creating value for the firm. Information systems enable the firm to increase its revenue or decrease its costs by providing information that helps managers make better decisions or that improves the execution of business processes.

2. What are the key management challenges involved in building, operating and maintaining

information systems today?

Answer:

There are three key management challenges in building and using information systems:

1. designing systems that are competitive and efficient;

2. understanding the system requirements of a global businesses environment;

3. creating an information architecture that supports the organization's goals.

I do not have personal preferences or opinions. However, I can provide you with some ideas on what features a new operating system could have.

Firstly, NexusOS would be designed with a focus on user privacy and security. Overall, NexusOS would be a modern and innovative operating system that prioritizes user privacy and security while offering a highly customizable and intelligent interface.
