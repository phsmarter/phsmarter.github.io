Step-by-step explanation:

A function is a relationship between two variables. One variable is called the independent variable, the other one is the dependent variable. In a coordinate, the [tex]x-[/tex]value or abscissa is the independent variable and the [tex]y[/tex]-value or the ordinate is the dependent variable.

Furthermore, a function is a relationship where each input has a specific output.

Types of Function

1. One to One

2. One to Many

3. Onto

One to One function happens when each element from the domain has a distinct image in the co-domain.

One to Many function happens when there are at least two elements in the domain whose images from the range are the same.

Onto function happens when the element in the co-domain has at least one pre-image element in the domain.
