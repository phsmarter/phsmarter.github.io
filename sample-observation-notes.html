22. SEATWORK:A. Look for: Margin of error, (E) Prepare the table for the Confidence level, a/2 & z Draw the standard normal curve Conclusion Formula: E = z(a/2) * σ / sqrt n 1. σ  = 0.3, n = 45, confidence level 98% 2. σ  = 0.5, n = 50, confidence level 98% 3. σ  = 1.5, n = 70, confidence level 98%ASSIGNMENT: B. Word problem: Joel wants to replicate a certain study, where the lowest observed value is 10.4 while the highest is 12.8. She wants to estimate the population mean µ to within an error of 0.05 of its true value. Using 98% confidence level find the sample size n that she needs. Notes: R (range) = Highest Observed Value - Lowest Observed Value Standard Deviation = σ = R/4  where:  R = Range Always Round up the n (Sample) ​


Answer/Step-by-step explanation:

A.

To find the margin of error (E), we can use the formula:

E = z(a/2) * σ / sqrt(n)

where z(a/2) is the z-value for the given confidence level and σ is the population standard deviation.

For a 98% confidence level, a/2 = 0.01, so z(a/2) = 2.33.

E = 2.33 * 0.3 / sqrt(45) = 0.155 (to three decimal places)

E = 2.33 * 0.5 / sqrt(50) = 0.330 (to three decimal places)

E = 2.33 * 1.5 / sqrt(70) = 0.412 (to three decimal places)

Therefore, the margin of error for the three cases are: 0.155, 0.330, and 0.412.

B.

We can use the formula for sample size:

n = [z(a/2) * σ / E]^2

where z(a/2) is the z-value for the given confidence level, σ is the population standard deviation (which we don't know), and E is the desired margin of error.

We can estimate the standard deviation using the range:

R = highest observed value - lowest observed value = 12.8 - 10.4 = 2.4

σ = R / 4 = 2.4 / 4 = 0.6

Substituting the values into the formula, we get:

n = [2.33 * 0.6 / 0.05]^2 = 678.98

We always round up the sample size, so Joel needs a sample size of at least 679.
