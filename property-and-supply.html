28. B. Supply the missing term to complete each statement using the given property. 1. 3 x (b x c) Associative Property 2. 5a + 3b. Commutative Property. 3. 15 + ___= 15 Identity Property 4. 23 + ____= 0 Inverse Property on Addition​


Answer and step-by-step explanation:

The given statements show the properties of real numbers. It will help us when we solve math problems.

1. 3 × (b × c) = (3 × b) × c

a × (b × c) = (a × b) × cIt states that if you regroup the numbers when multiplying, the product will not change.

2. 5a + 3b = 3b + 5a

a + b = b + aIt states that if you switch the numbers when adding, the sum will not change.

3. 15 + 0 = 15

a + 0 = aIt states that if you add any number to zero, the answer will always be the number.

4. 23 + (-23) = 0

a + (-a) = 0It states that if you add the two same positive and negative numbers, the answer will always be zero.
