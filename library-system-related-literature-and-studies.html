17. Enumerate the 2 Library Classification Systems.​


Answer:

2 Main Library Classification Systems:The Dewey Decimal System

➡️ first classifies books into nine broad categories, each identified by a number in the hundreds.

The Library of Congress System

➡️ has 21 broad categories, each represented by one letter of the alphabet.

Explanation:

The Library of Congress system is an outline, much like one you would use to organize an essay. Subtopics are dealt with by indentation. The Dewey Decimal system is a hierarchy: Subtopics are dealt with by using hundreds, then tens, then units and then decimal places.

The Dewey Decimal System is used by more libraries, but the Library of Congress system is used by some of the biggest libraries -- most obviously the Library of Congress itself. Many universities use the Library of Congress system, but public libraries mostly use the Dewey Decimal System.


18. differentiate library of congress system from the dewey decimal system​


Answer:

The main difference between Library of Congress System and Dewey Decimal System is that Library of Congress System has 21 classes, allowing for more call numbers, while Dewey Decimal System only has 10 classes, allowing only a limited number of call numbers.

20. give the background of the study about library management: Navigating books at the library


Answer:

The study of library management and book navigation has been going on for centuries. To assist users in gaining access to materials, librarians and information scientists have developed tools and techniques such as classification systems and digital search tools. Technological advancements have also opened up new possibilities for library management and user navigation. This field of study remains vital as libraries adapt to changing user needs and technological advances while remaining true to their core mission of providing access to knowledge and information.
